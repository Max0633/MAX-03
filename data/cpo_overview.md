# CPO（共封装光学）技术概述

> **最后更新时间：** 2026年5月25日
> **数据截止：** 2026年Q2

## 1. 技术定义

**共封装光学（Co-Packaged Optics，CPO）** 是一种将光学引擎（光收发器）直接集成到交换芯片或计算芯片封装内的技术。与传统的可插拔光模块不同，CPO将光电转换功能靠近芯片核心，显著缩短了电信号传输距离。

**核心原理：**
- 将硅光子芯片（Silicon Photonics）与电子芯片（如交换ASIC）在同一封装内集成
- 采用先进封装技术（如2.5D/3D封装、扇出型晶圆级封装等）
- 实现光电协同设计，优化信号完整性和功耗

## 2. 与传统光模块的对比优势

| 特性 | 传统可插拔光模块（如QSFP-DD） | CPO共封装光学 |
|------|-----------------------------|--------------|
| **功耗** | 较高（长SerDes通道损耗） | 降低30-50%（短电互连） |
| **带宽密度** | 受前面板空间限制 | 显著提高（突破面板限制） |
| **延迟** | 较高（长电信号路径） | 降低（短光电路径） |
| **信号完整性** | 受限于长走线 | 大幅改善 |
| **可维护性** | 热插拔，易于更换 | 难以现场维修 |
| **标准化** | 成熟（MSA标准） | 逐步完善（OIF标准） |
| **成本** | 规模化后成本较低 | 成本逐步下降 |
| **应用成熟度** | 广泛部署 | 2025年底开始规模商用 |

## 3. 主要应用场景

### 3.1 数据中心
- 超大规模数据中心的高带宽互连
- 支持102.4T及以上交换容量
- AI/ML训练集群的高吞吐量互连

### 3.2 AI计算
- 大语言模型（LLM）训练和推理的高速互连
- GPU/CPU集群间的光互连
- 支持1.6T/3.2T高速率

### 3.3 5G和边缘计算
- 5G前传/中传/回传网络
- 边缘数据中心的高密度互连
- 低延迟、高带宽需求

### 3.4 高性能计算（HPC）
- 超级计算机的光互连
- 科学计算和仿真应用

## 5. 技术路线之争（2026年现状）

### 5.1 硅光方案（Silicon Photonics）
- **优势**：利用成熟的CMOS制造工艺，成本下降明显
- **挑战**：III-V族激光器集成仍有挑战
- **代表厂商**：Intel、Cisco/Acacia、GlobalFoundries、TSMC
- **2026年进展**：已成为CPO主流技术路线，多家厂商实现量产

### 5.2 InP方案
- **优势**：调制效率更高，激光集成度好
- **挑战**：制造成本仍较高
- **代表厂商**：InPACT、Smart Photonics、POET Technologies
- **2026年进展**：在特定应用场景保持竞争力

### 5.3 混合集成方案
- **优势**：结合硅光与InP优势
- **现状**：主流厂商普遍采用此方案
- **代表厂商**：Broadcom、中际旭创等

## 6. CPO vs NPO（2026年对比）

| 特性 | CPO | NPO |
|------|-----|-----|
| 光引擎位置 | 直接集成在ASIC封装上 | 放置在ASIC封装附近 |
| 电学距离 | 极短 | 约10-25mm |
| 可维护性 | 改善中 | 较好 |
| 技术成熟度 | 商用阶段 | 过渡方案 |
| 标准化 | OIF标准完善 | OIF标准完善 |
| **2026年状态** | **规模商用** | **仍有应用** |
