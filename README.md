# CPO共封装光学信息收集与分析项目

## 项目概述
本项目旨在全面收集关于CPO（Co-Packaged Optics，共封装光学）的行业信息，并进行系统性分析，为投资决策提供数据支持。

## 信息收集范围
- CPO技术概述与原理
- 市场规模与增长预测
- 产业链分析
- 主要公司与竞争格局
- 投资价值分析

## 项目结构
```
CPO_analysis/
├── README.md                    # 项目说明
├── data/
│   ├── cpo_overview.md          # CPO技术概述
│   ├── market_analysis.md       # 市场分析数据
│   ├── industry_chain.md        # 产业链分析
│   ├── companies.md             # 主要公司信息
│   └── investment_analysis.md   # 投资分析
├── reports/
│   └── cpo_analysis_report.md   # 综合分析报告
└── scripts/
    └── data_analysis.py         # 数据分析脚本
```

## 使用方法
1. 浏览`data/`目录下的各个数据文件
2. 查看`reports/cpo_analysis_report.md`获取综合分析
3. 运行`scripts/data_analysis.py`进行数据分析

## 数据来源
- 行业研究报告
- 公司财报与公告
- 新闻媒体报道
- 专业数据库
